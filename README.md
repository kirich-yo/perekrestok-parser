## Необходимые модули

requests, BeautifulSoup, openpyxl, pyfiglet, rich

Их можно установить, выполнив следующие команды:

```shell
pip install requests
pip install beautifulsoup4
pip install openpyxl
pip install pyfiglet
pip install rich
```
